Neighborhood Map

A single-page application using KnockoutJS with a map of neighborhood, including map markers for locations, a search function to discover these locations, and a listview to support simple browsing of all locations. Also added few third-party APIs like Four Square, Wikipedia, etc. to provide additional information about these locations.

See Project Live
Github repo
Summary
  • A responsive single-page Google Maps API based application using KnockoutJS
  • The map of neighborhood includes map markers for locations and a listview to support simple browsing of all location
  • Implemented third-party StreetView API to provide nearby imagery within the map as an overlay into a infowindow
  • Retrieved information about the selected location using the Foursquare API
  • Also, fetched images related to the selected location from the FourSquare API
